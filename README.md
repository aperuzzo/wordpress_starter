This is a wordpress theme starter theme combining the _tk theme with a grunt build

The starter theme _tk was used as a foundation: https://github.com/Themekraft/_tk/

The grunt build includes sass, concat, watch

if you are looking for a starter worpress theme that entegrates bootstrap and grunt, try this out!


Dependancies:

Grunt.js http://gruntjs.com/
Node.js https://nodejs.org/en/
Sass http://sass-lang.com/

1. make sure node.js and grunt.js are installed globally on your machine
2. in the command line, go to your project folder
3. npm init
4. npm install grunt --save dev
5. install grunt plugins:

npm install grunt-contrib-concat --save-dev

npm install grunt-contrib-sass --save-dev

npm install grunt-contrib-watch --save-dev

6. install sass (on mac):

sudo gem install sass
